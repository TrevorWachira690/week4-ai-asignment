## What is Data?
Data is raw facts, numbers, words and figures that can be stored and processed to provide information.
## Explain the difference between:
- **Structured Data**- Structured data is data that has a fixed schema, for example: *phone numbers* and *names*.
- **Unstructured Data**- Unstructured data is data that does not have a fixed schema, for example: *Videos* and *images*.
## What is a schema?
A schema is a structure that defines the format of data in a system.
